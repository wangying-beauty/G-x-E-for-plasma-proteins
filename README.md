# Gene-Environment Interactions for Plasma Proteins

Overview
This project provides a comprehensive computational workflow including:
1. Protein Data Quality Control (UK Biobank Pharma Proteomics Project)
2. Variance Quantitative Trait Locus (vQTL) discovery
3. Gene-Environment Interaction (G×E) screening using OSCA and PLINK2
4. Independent mediation analysis (Genotype-stratified models)
5. Disease risk prediction using LightGBM
